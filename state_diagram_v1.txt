      [main.py]
  -> checks market
  -> sends entry alert
  -> trader decides manually

[trader]
  -> enters trade manually
  -> writes / updates active_trade.json

[trade_monitor.py]
  -> watches active trade
  -> checks current price every N seconds
  -> warns trader in Thai:
     HOLD
     WATCH CLOSELY
     TAKE SMALL PROFIT
     EXIT NOW

[key philosophy]
  -> system is not auto-trading
  -> system is an alert assistant
  -> TP is a reference, not mandatory every time
  -> protecting profit before reversal is a core goal
